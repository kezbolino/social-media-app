How do I find this code? (30 seconds)
  1. Open instagram.com in this browser and make sure you're logged in.
  2. Press ⌥⌘I (Mac) or F12 (Windows) to open the developer panel.
  3. Click the Application tab. In the left sidebar open Storage → Cookies → https://www.instagram.com.
  4. Find the row named sessionid and copy its long Value.
  5. Paste it in the box above. (It's remembered on this computer so you only do this once.)

If results stop working later, your code expired — just grab a fresh one the same way.
